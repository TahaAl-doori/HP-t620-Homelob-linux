# HP-t620-Homelob-linux
Project: HomeLab-Setup on HP t620

Overview: I successfully transformed a thin client into a functional Linux home server using KVM/QEMU

Technical Accomplishments:

OS:Linux-Mint 22(Host) / Ubuntu Server24.04 (Guest)
Optimization: Configured dynamic memory allocation (2GB-3GBRAM) to ensure system stability on limited hardware
Remote Management: Monitoring system health and service via Cockpit Web Console
Troubleshooting: Resolved hardware resource conflicts and browser compastibility issues (Switching to Firefox for stability)
